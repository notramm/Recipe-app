## Taste-Trail

A react (`Next.js` + `React Query`) app, where you can find meals recipe and save your favorite recipe for later.

### Tech Used

- Next.js
- React Query
- Axios
- React-hot-toast

**live preview: [https://meal-khuj.netlify.app](https://meal-khuj.netlify.app)**<br>
